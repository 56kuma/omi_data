# Git 活用とJSON基盤アプリ導入の検討

開発メンバー間で、現在の業務フローやデータ管理の問題点と、その改善策について議論している。既存の手作業（コピペ運用やローカルでの都度対応）がボトルネックになっており、マスターデータをJSONで定義し、それを読み込ませる専用アプリを作ることで、入力・出力・共有を自動化し、利用者も含めて全員が楽になる構想を話している。また、OpenAI系ツール（o1、Sonnet 3.5 など）を設計と実装で使い分ける効率的なワークフローにも言及している。さらに、矢部さんにGitを本格的に使ってもらうべきタイミングだという話から、Git Immersionのような学習ツールの紹介、ブランチ運用や並行作業のやり方などGitの概念的な説明も交えつつ、今後の開発体制・環境整備の方向性を確認している。
